Detailed Description

A metric graph class.

Metric graph is defined as a graph \((V,E)\) with each edge \(e \in E\) being associated with a certain interval \([0,l(e)]\), where \(l(e) > 0\).

Updates edge characteristics

This function adds a new edge or updates already existing one. The exact behaviour depends on the parameters and prior configuration of graph. Say, we want to add an edge of length \(l\) from vertex \(v\) to vertex \(w\). Possible cases are:

Is \(v \in V\)? Is \(w \in V\)? Type of existing edge Desired type of edge Result
No Yes None Any All non-existing vertices are added to the graph (i.e., \(V = V \cup \{v,w\}\)) together with desired edge of length \(l\).
Yes No
No No
Yes Yes None Any Desired edge from \(v\) to \(w\) of length \(l\) is added to the graph.
\(v \rightarrow w\) \(v \rightarrow w\) The length of existing edge \(v \rightarrow w\) is updated to be \(l\).
\(v-w\) Existing edge \(v \rightarrow w\) is replaced with \(v-w\) of length \(l\).
\(v \leftarrow w\) \(v \rightarrow w\) Existing edge \(v \leftarrow w\) is replaced with \(v-w\) of length \(l\).
\(v-w\) Existing edge \(v \leftarrow w\) is replaced with \(v-w\) of length \(l\).
\(v-w\) \(v \rightarrow w\) The length of existing edge \(v-w\) is updated to be \(l\).
\(v-w\) The length of existing edge \(v-w\) is updated to be \(l\).
Parameters
out_vertexID of vertex \(v\).
in_vertexID of vertex \(w\).
lengthDesired length of edge between \(v\) and \(w\) (i.e., \(l\)).
is_directedIf true, desired type of edge will be \(v \rightarrow w\), if false, desired type of edge will be \(v-w\).
